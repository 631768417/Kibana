Installation

No templates to apply, no SQL files to import.

We tried to keep this simple. You've already setup and tuned Logstash and Elasticsearch, this is the easy part. Kibana really only requires a browser, a webserver, PHP and one very common PHP module.

1. Download from github

Or head over to the Github project page

Github Project Page »

2. Install

You've made it this far. I'm going to bet you know how to ...

  1. Install the PHP5 curl module (php5-curl) on your webserver
  2. Upload the zip or tarball to your webserver
  3. Extract it
  4. Rename the extracted directory to something useful, for example 'kibana'
  5. Copy that directory into your webserver's DocumentRoot
Whooaa! Read the configuration section to the right before pointing your web browser at your shiney new installation

3. Configure one thing

Almost there, just one required thing in config.php

Change elasticsearch1 to point at the host with your elasticsearch installation.

'elasticsearch_server' => "elasticsearch1:9200",

Heads up! Note the , (comma) at the end of the line, that's important.

And you're done!

Point your web browser over to where ever you installed Kibana and poke around. For more information, including a short introduction to Lucene search syntax, check out the about section of this page.

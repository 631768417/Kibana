Support

Questions, comments, suggestions?

Github

The Kibana project is based out of Github. You can find the source, send me a message, file a bug report or submit a feature request at our Github Project Page »

Submit an issue

If you think you've found a bug, you probably have. When submitting a bug or feature request please be as detail oriented as possible. If submitting a bug include all steps required to reproduce. Indicate versions for: php, elasticsearch, logstash, your browser. Kibana issues »

Fork me Do better than a bug report

Knowing something is broken sucks. Knowing something is fixed is awesome. Consider forking Kibana on Github and submitting a pull request. Patches in diff formatting are cool too!
